# Images d'Équipement

Ce dossier contient les images des équipements utilisés par EPS.

## Structure des fichiers

Pour chaque type d'équipement, ajoutez 3 images (format JPG recommandé) :

### 1. Monobrosse
- `monobrosse-1.jpg` - Vue principale de la monobrosse
- `monobrosse-2.jpg` - Monobrosse en action
- `monobrosse-3.jpg` - Détail de la machine

### 2. Autolaveuse
- `autolaveuse-1.jpg` - Vue principale de l'autolaveuse
- `autolaveuse-2.jpg` - Autolaveuse en opération
- `autolaveuse-3.jpg` - Détail de l'équipement

### 3. Chariot Professionnel
- `chariot-1.jpg` - Chariot professionnel général
- `chariot-2.jpg` - Chariot toilettes
- `chariot-3.jpg` - Chariot bureaux

### 4. Aspirateur Multifonction
- `aspirateur-1.jpg` - Vue principale de l'aspirateur
- `aspirateur-2.jpg` - Aspirateur en utilisation
- `aspirateur-3.jpg` - Détail de l'aspirateur

### 5. Camion Balayeur
- `camion-1.jpg` - Vue principale du camion balayeur
- `camion-2.jpg` - Camion balayeur en action
- `camion-3.jpg` - Détail du camion

## Spécifications recommandées

- **Format** : JPG ou PNG
- **Dimensions** : 800 x 600 px (format paysage)
- **Poids** : < 300 KB par image
- **Qualité** : 80-85%

## Note

Si une image n'est pas disponible, le système utilisera automatiquement une image de fallback (`/images/services/nettoyage.jpg`).
