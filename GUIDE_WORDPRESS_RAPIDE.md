# Guide Rapide - Ajouter un Témoignage Client

Ce guide vous montre comment ajouter rapidement un témoignage client via WordPress.

## Étapes rapides

### 1. Connexion à WordPress

1. Allez sur `https://votre-site-wordpress.com/wp-admin`
2. Connectez-vous avec vos identifiants

### 2. Ajouter un nouveau témoignage

1. Dans le menu latéral, cliquez sur **Témoignages**
2. Cliquez sur **Ajouter**

### 3. Remplir les informations

#### Titre de la page (titre WordPress)
Entrez un titre descriptif, par exemple : "Témoignage - Ahmed Ould Mohamed"

#### Champs du témoignage

**Nom complet** (Requis)
```
Ahmed Ould Mohamed
```

**Poste/Fonction** (Requis)
```
Directeur Général
```

**Entreprise** (Requis)
```
Centre Commercial Al-Khaima
```

**Témoignage** (Requis)
Le texte complet du témoignage :
```
Un service impeccable ! L'équipe d'EPS a transformé nos locaux commerciaux.
Professionnalisme et efficacité au rendez-vous. Nous recommandons vivement leurs services.
```

**Note** (Optionnel)
Sélectionnez une note de 1 à 5 (généralement 5 pour les témoignages positifs)

**Avatar** (Optionnel)
- Cliquez sur "Ajouter une image"
- Uploadez la photo du client
- Ou laissez vide pour afficher les initiales automatiquement

**Date** (Optionnel)
Sélectionnez la date du témoignage

### 4. Publier

1. Dans la colonne de droite, cliquez sur **Publier**
2. Le témoignage apparaîtra automatiquement sur le site dans l'heure qui suit (selon la configuration du cache)

## Conseils pour de bons témoignages

### Longueur du témoignage
- **Optimal** : 2-3 phrases (environ 100-150 mots)
- Trop court : Moins crédible
- Trop long : Difficile à lire

### Contenu du témoignage
Incluez :
- ✅ Le problème ou besoin initial
- ✅ La solution apportée par EPS
- ✅ Les résultats obtenus
- ✅ Une recommandation

### Informations du client
- Utilisez le nom complet réel du client
- Incluez un poste/fonction précis
- Mentionnez l'entreprise complète
- Ajoutez une photo si possible (avec autorisation)

---

## Gérer les témoignages existants

### Modifier un témoignage

1. Allez dans **Témoignages**
2. Survolez le témoignage
3. Cliquez sur **Modifier**
4. Faites vos modifications
5. Cliquez sur **Mettre à jour**

### Supprimer un témoignage

1. Allez dans **Témoignages**
2. Survolez le témoignage
3. Cliquez sur **Mettre à la corbeille**

### Ordre d'affichage

Les témoignages sont affichés par ordre de publication (plus récent en premier).

---

## Vérifier l'affichage

1. Attendez 1 heure (temps de cache) ou redémarrez le serveur en développement
2. Allez sur la page d'accueil du site : `https://eps-mauritanie.com`
3. Scrollez jusqu'à la section "Témoignages"
4. Votre nouveau témoignage devrait apparaître

## Questions fréquentes

**Q : Combien de témoignages puis-je ajouter ?**
R : Autant que vous voulez. Ils s'afficheront tous dans un slider ou une grille.

**Q : Puis-je modifier un témoignage après publication ?**
R : Oui, les modifications seront visibles après le temps de cache (1 heure).

**Q : Que faire si la photo ne s'affiche pas ?**
R : Vérifiez que l'image est au format JPG ou PNG et fait moins de 2 MB.

**Q : Le témoignage n'apparaît pas sur le site**
R : Vérifiez que le statut est "Publié" et non "Brouillon". Attendez 1 heure pour le cache.
